随机梯度下降

随机梯度下降（Stochastic Gradient Descent, SGD）是一种常用的优化算法，用于最小化目标函数。我们可以用蒙特卡洛方法的思想来理解随机梯度下降。
神经网络的训练可以定义为这样的优化问题：

min⁡wEX∼p(⋅)[L(X;w)]\min_{w} \mathbb{E}_{X \sim p(·)}[L(X;w)] wmin​EX∼p(⋅)​[L(X;w)]

其中，www是神经网络的参数，p(⋅)p(·)p(⋅)是输入数据的分布，LLL是损失函数。
目标函数的梯度\mathbb{E}_{X \sim p(·)}[L(X;w)]关于参数www的梯度可以表示为：

g≜∇wEX∼p(⋅)[L(X;w)]=EX∼p(⋅)[∇wL(X;w)]g \triangleq \nabla_w \mathbb{E}_{X \sim p(·)}[L(X;w)] = \mathbb{E}_{X \sim p(·)}[\nabla_w L(X;w)] g≜∇w​EX∼p(⋅)​[L(X;w)]=EX∼p(⋅)​[∇w​L(X;w)]

可以做梯度下降更新w，以减小目标函数的值：

w←w−αgw \leftarrow w - \alpha g w←w−αg

其中，α\alphaα是学习率。直接计算全部样本的梯度g的期望会非常耗时，因此做蒙特卡洛方法近似，把得到的近似梯度g^\hat{g}g^​作为w的更新量。

  1. 根据p(x)随机采样得到B个样本，记作x^1,x^2,…,x^B\hat{x}_1, \hat{x}_2, \ldots, \hat{x}_Bx^1​,x^2​,…,x^B​
  2. 计算每个样本的梯度g^i=∇wL(x^i;w)\hat{g}_i = \nabla_w L(\hat{x}_i;w)g^​i​=∇w​L(x^i​;w)
  3. 计算B个样本的梯度的平均值g^=1B∑i=1Bg^i\hat{g} = \frac{1}{B} \sum_{i=1}^B \hat{g}_ig^​=B1​∑i=1B​g^​i​
  4. 更新w：w←w−αg^w \leftarrow w - \alpha \hat{g}w←w−αg^​
    样本数量B称作batch size（批大小），通常是比较小的正整数，比如1、8、16、32等。所以称之为小批量SGD。
    在实际应用中，p(x)一般未知，训练神经网络时通常收集一个训练集X=x1,x2,…,xn\mathcal{X} = {x_1, x_2, \ldots, x_n}X=x1​,x2​,…,xn​，并求解下面这样一个经验风险最小化问题：

min⁡w1n∑i=1nL(xi;w)\min_{w} \frac{1}{n} \sum_{i=1}^n L(x_i;w) wmin​n1​i=1∑n​L(xi​;w)

这相当于用下面这个概率密度函数来代替 p(x)p(x)p(x)：

p(x)={1nif x∈X0otherwisep(x) = \begin{cases} \frac{1}{n} & \text{if } x \in \mathcal{X} \\ 0 & \text{otherwise} \end{cases} p(x)={n1​0​if x∈Xotherwise​

强化学习基本概念

回合episode和epoch的区别

  • episode
    回合指示一个完整的交互过程，包括一系列的状态、动作和奖励。一个回合可以包含多个交互步骤，直到达到终止状态或达到最大步数。就是打完一把游戏。
  • epoch
    一个epoch表示在整个训练数据集上进行一次完整的前向传播和反向传播过程。一个epoch通常包含多个回合，每个回合都包含多个交互步骤。

回报return

return是从当前时刻开始，到回合结束的所有奖励之和，也叫为累计奖励。
强化学习的目标是最大化回报，而不是最大化当前奖励。

折扣率γ\gammaγ

使用折扣率小于1，无限期MDP的回报是有界的。

动作价值函数Qπ(s,a)Q_{\pi}(s,a)Qπ​(s,a)

动作价值函数是指在给定状态sss和动作aaa的情况下，根据策略π\piπ，未来可能获得的累计奖励的期望值。

Qπ(s,a)=Eπ[Rt+1+γRt+2+⋯∣St=s,At=a]Q_{\pi}(s,a) = \mathbb{E}_{\pi}[R_{t+1} + \gamma R_{t+2} + \cdots | S_t = s, A_t = a] Qπ​(s,a)=Eπ​[Rt+1​+γRt+2​+⋯∣St​=s,At​=a]

最优动作价值函数Q∗(st,at)Q^*(s_t,a_t)Q∗(st​,at​)

最优动作价值函数是指在给定状态sts_tst​和动作ata_tat​的情况下，在最优策略下未来可能获得的累计奖励的期望值。

Q∗(st,at)=max⁡aQπ(st,a)Q^*(s_t,a_t) = \max_{a} Q_{\pi}(s_t,a) Q∗(st​,at​)=amax​Qπ​(st​,a)

意思就是有多种策略π\piπ可以选择，选择最优策略π∗\pi^*π∗，在最优策略π∗\pi^*π∗下未来可能获得的累计奖励的期望值。

状态价值函数Vπ(s)V_{\pi}(s)Vπ​(s)

状态价值函数是指根据策略π\piπ,在给定状态sss的情况下，未来可能获得的累计奖励的期望值。

Vπ(st)=Eπ[Qπ(st,at)∣St=st]=∑a∈Aπ(a∣st)Qπ(st,a)V_{\pi}(s_t) = \mathbb{E}_{\pi}[Q_{\pi}(s_t,a_t) | S_t = s_t] = \sum_{a \in \mathcal{A}} \pi(a|s_t) Q_{\pi}(s_t,a) Vπ​(st​)=Eπ​[Qπ​(st​,at​)∣St​=st​]=a∈A∑​π(a∣st​)Qπ​(st​,a)

只依赖于状态sts_tst​，不依赖于动作ata_tat​。也是回报的期望值。

DQN与Q-Learning

用Q∗Q^*Q∗最优动作价值函数使用最大化消除策略π\piπ。
可以这样理解Q∗Q^*Q∗：已知状态sts_tst​和动作ata_tat​，不论未来采取什么样的策略π\piπ，回报的期望不可能超过Q∗Q^*Q∗。
所以要训练一个近似的Q∗Q^*Q∗。

深度Q网络（Deep Q Network, DQN）

记作Q(s,a;w)，其结构如下：

  1. 输入：状态sss
  2. 卷积层：提取特征
  3. 全连接层：映射到动作空间
  4. 输出：离散动作空间A\mathcal{A}A上的每个动作aaa的Q值，即输出是一个A\mathcal{A}A维的向量

DQN的梯度：

∇wQ(s,a;w)≜∂Q(s,a;w)∂w\nabla_w Q(s,a;w) \triangleq \frac{\partial Q(s,a;w)}{\partial w} ∇w​Q(s,a;w)≜∂w∂Q(s,a;w)​

TD算法

训练DQN最常用的算法是TD（Temporal Difference, 时序差分）算法。
损失函数：

L(w)=12[Q(s,a;w)−y]2L(w) = \frac{1}{2}[Q(s,a;w) - y]^2 L(w)=21​[Q(s,a;w)−y]2

梯度：

∇wL(w)=∇w12[Q(s,a;w)−y]2=(q^−y)⋅∇wQ(s,a;w)\nabla_w L(w) = \nabla_w \frac{1}{2} [Q(s,a;w) - y]^2 = (\hat{q} - y)·\nabla_w Q(s,a;w) ∇w​L(w)=∇w​21​[Q(s,a;w)−y]2=(q^​−y)⋅∇w​Q(s,a;w)

然后梯度下降更新w。
TD算法就是把预测到中途的数据也利用上。
把到中途的q记作r（训练时是一个状态上的奖励），中途之后的路程q^\hat{q}q^​，则TD目标为y^≜r+q^\hat{y} \triangleq r + \hat{q}y^​≜r+q^​，新的梯度为：

∇wL(w)=(q^−y^)⋅∇wQ(s,a;w)\nabla_w L(w) = (\hat{q} - \hat{y})·\nabla_w Q(s,a;w) ∇w​L(w)=(q^​−y^​)⋅∇w​Q(s,a;w)

δ=q^−y^\delta = \hat{q} - \hat{y}δ=q^​−y^​ 称为TD误差。做一次梯度下降更新w：

w←w−α∇wL(w)=w−α(q^−y^)⋅∇wQ(s,a;w)=w−α⋅δ⋅∇wQ(s,a;w)w \leftarrow w - \alpha \nabla_w L(w) = w - \alpha (\hat{q} - \hat{y})·\nabla_w Q(s,a;w) = w - \alpha · \delta ·\nabla_w Q(s,a;w) w←w−α∇w​L(w)=w−α(q^​−y^​)⋅∇w​Q(s,a;w)=w−α⋅δ⋅∇w​Q(s,a;w)

TD误差δ\deltaδ就是模型的估计与真实值之间的差距。

用TD算法训练DQN

近似可得：

Q∗(st,at)≈rt+γ⋅max⁡a∈AQ∗(st+1,a)Q^*(s_t,a_t) \approx r_t + \gamma · \max_{a \in \mathcal{A}}Q^*(s_{t+1},a) Q∗(st​,at​)≈rt​+γ⋅a∈Amax​Q∗(st+1​,a)

把上式中的Q∗(⋅)Q^*(·)Q∗(⋅)替换为神经网络Q(s,a;w)Q(s,a;w)Q(s,a;w)，则有：

Q(s,a;w)≈rt+γ⋅max⁡a∈AQ(st+1,a;w)Q(s,a;w) \approx r_t + \gamma · \max_{a \in \mathcal{A}}Q(s_{t+1},a;w) Q(s,a;w)≈rt​+γ⋅a∈Amax​Q(st+1​,a;w)

做梯度下降：

w←w−α⋅δt⋅∇wQ(st,at;w)w \leftarrow w - \alpha · \delta_{t} ·\nabla_w Q(s_t,a_t;w) w←w−α⋅δt​⋅∇w​Q(st​,at​;w)

训练流程

收集训练数据
用任意策略π\piπ与环境交互，收集训练数据，此时称为行为策略π\piπ。比较常用的是ϵ\epsilonϵ-贪婪策略。

at={argmaxaQ(st,a;w)with probability 1−ϵA中的随机动作with probability ϵa_t = \begin{cases} argmax_{a} Q(s_t,a;w) & \text{with probability } 1 - \epsilon \\ \mathcal{A} 中的随机动作 & \text{with probability } \epsilon \end{cases} at​={argmaxa​Q(st​,a;w)A中的随机动作​with probability 1−ϵwith probability ϵ​

把一个回合的数据称为一个轨迹，记作：

T={s1,a1,r1,s2,a2,r2,⋯ ,sT,aT,rT}\mathcal{T} = \{s_1,a_1,r_1,s_2,a_2,r_2,\cdots,s_T,a_T,r_T\} T={s1​,a1​,r1​,s2​,a2​,r2​,⋯,sT​,aT​,rT​}

把一条轨迹分为n个(st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st​,at​,rt​,st+1​)这样的四元组，存入缓存，这个缓存称为回放缓冲区。

更新参数w

  1. 对DQN做正向传播，得到Q值：

q^j+1=max⁡a∈AQ(sj+1,a;wnow)\hat{q}_{j+1} = \max_{a \in \mathcal{A}} Q(s_{j+1},a;w_{now}) q^​j+1​=a∈Amax​Q(sj+1​,a;wnow​)

  1. 计算TD目标和TD误差：
